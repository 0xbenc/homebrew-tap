# 0xbenc Homebrew Tap

Homebrew formulas for 0xbenc projects

## passage

Install:

```sh
brew tap 0xbenc/tap
brew install --cask passage
```

Or install directly:

```sh
brew install --cask 0xbenc/tap/passage
```

## ssherpa

Install:

```sh
brew tap 0xbenc/tap
brew install --cask ssherpa
```

Or install directly:

```sh
brew install --cask 0xbenc/tap/ssherpa
```
